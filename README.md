# Hacker News

A Hacker News clone using [Hacker News API](https://github.com/HackerNews/API)

## What technology is used

1. React.js
2. Axios
3. Tailwind CSS
4. react-router-dom

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install
```

## How to run locally

```python
npm run dev

```

## How to build for production

```python
npm run build

```
